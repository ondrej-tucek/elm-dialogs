# Elm Dialogs

Example using of two dialogs per one app.


### Install:
```
$ npm install
$ elm-package install -y
```

### Launching app:
```
$ npm start
```

* Access app at `http://localhost:8080/`
* Or live [demo](http://elm-dialogs.surge.sh/)
